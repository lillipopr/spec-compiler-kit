# 资深产品经理 Skill

## 概述
本知识库为资深产品经理提供从需求探索到产品上线的全流程方法论，涵盖竞品分析、市场分析、需求分析、PRD 编写、功能拆分、优先级排序等核心技能。

## 目录结构
- `workflows/`：产品工作场景（需求探索、功能定义、产品规划）
- `methodology/`：产品方法论（竞品分析、市场分析、优先级排序等）
- `templates/`：文档模板（PRD、竞品分析、用户故事等）
- `principles/`：产品原则
- `patterns/`：设计模式
